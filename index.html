Phoenix Framework

CSS modularized simple toolkit for beautiful websites.

* This framework is still under construction, any feedback is very welcome :)

Current Features

  • Responsive design
  • Custom classes to perform fast changes (background, font and border colors)
  • Responsive grid, 6 levels
  • Modularized in components
  • Very simple design, perfect for light landpages

Forms

Phoenix Framework offers multiple form combinations you can use easily

Code Example:


<form class="pf-form pf-inline">
  <label class="pf-label">Name</label>
    <section class="pf-inline-form">
      <input type="text" class="pf-input pf-inline w70 right  border-blue6">
      <button class="pf-button pf-form-button pf-inline w30 right bg-blue3 border-blue6">Click</button>
    </section>
  <label class="pf-label">Search</label>
    <section class="pf-inline-form">
      <button class="pf-button pf-form-button pf-inline w30 left"> Click </button>
      <input type="text" class="pf-input pf-inline w70 left">
    </section>
    <label class="pf-label">Email</label>
      <input type="email" class="pf-input pf-input w100">
      <label class="pf-label">Car</label>
    <select class="pf-select bg-orange3">
      <option value="volvo">Volvo</option>
      <option value="saab">Saab</option>
      <option value="mercedes">Mercedes</option>
      <option value="audi">Audi</option>
    </select>
</form>
